# RFP Response Generator

An advanced AI-powered RFP Response Generator that leverages multi-model vector search and intelligent requirement matching to streamline proposal creation, with enhanced collaboration and adaptive learning capabilities.

## Key Features

- Multi-AI model integration (OpenAI, Anthropic, Deepseek, Mixture of Agents)
- Intelligent vector search for similar requirement matching
- Mobile-responsive design with optimized UI
- Excel export with formatted output
- Categorized requirements with filtering and sorting
- Comprehensive reference tracking
- WhatsApp, Email and Markdown export capabilities
